A music player using two decks. The project was made using React, Bootstrap, SCSS, and React DnD.  
Songs can be loaded to the decks via context menu (on click) or drag and drop.  
The current state of the songs is kept on refresh / reload via local storage.

[Github repo](https://github.com/Toww/two-decks-player) / [Online version](https://two-decks-player.vercel.app/)
